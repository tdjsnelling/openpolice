About Open Police

Open Police is a web app developed by Tom Snelling. It is built on the data.police.uk API, which can be found here.

The service is open source; view the code at github.com/tdjsnelling/openpolice.

The following open source projects are used here:

  • JQuery
  • Bootstrap 4
  • bootstrap-datepicker
  • gmaps.js
  • jquery-locationpicker

OPENPOLICE.INFO

is a service that makes data on all street-level crimes more readily available.
To use, drag the marker to your desired location, select a date, and hit find. All locations are approximate.
